# Employee Attendance Management System

A database-driven web application designed to automate employee attendance and leave management. Built using PHP and MySQL, the system implements CRUD operations, admin authentication, and relational database design for secure and efficient data handling.

## Key Features
- Admin authentication and authorization
- Employee, department, attendance, and leave management
- Full CRUD operations
- Attendance search by employee ID and date
- Dashboard with summary statistics

## Technology Stack
- Backend: PHP
- Frontend: HTML, CSS
- Database: MySQL
- Server: XAMPP
- Tools: VS Code

## Database Design
- Normalized relational database
- Tables: Admin, Employee, Department, Attendance, Leave
- One-to-many relationships using foreign keys

## Setup Instructions
1. Install XAMPP and start Apache and MySQL
2. Copy the project folder to htdocs
3. Import the database into phpMyAdmin
4. Configure database credentials
5. Run the application at http://localhost/DBMS-mini-project/
